# Content Marketer — Content Strategy & Distribution Reference

Load when planning a content calendar, building a content program, or auditing existing content.

---

## Content Strategy Framework

```
1. AUDIENCE  → Who are we talking to? What do they search for / struggle with?
2. GOAL      → Awareness / Authority / Lead gen / Retention / SEO?
3. FORMAT    → Blog / Video / Podcast / Newsletter / Social?
4. CHANNEL   → Where does this audience already go?
5. CADENCE   → How often can we produce quality content sustainably?
6. MEASURE   → How do we know it's working? (traffic, leads, shares, backlinks)
```

---

## Content Types by Funnel Stage

| Stage | Content Type | Goal |
|---|---|---|
| Awareness (TOFU) | Blog posts, social, videos, podcasts | Reach new audience |
| Consideration (MOFU) | Guides, webinars, case studies, comparison pages | Build trust |
| Decision (BOFU) | Demo, pricing, ROI calculator, testimonials | Convert |
| Retention | Newsletter, in-app tips, release notes, community | Keep + expand |

## Blog Post Structures

### How-To Guide (evergreen, high SEO value)
```
Title: "How to [achieve outcome] [constraint]"
       "How to Set Up CI/CD with GitHub Actions in 20 Minutes"

Structure:
H1: [Title]
Intro: [Problem + what reader will learn]
H2: [Step 1 — concrete action]
H2: [Step 2]
H2: [Step 3]
H2: Common mistakes to avoid
Conclusion: [Summary + next step CTA]
```

### Listicle (shareable, fast to scan)
```
Title: "[N] [Adjective] [Things] [Context]"
       "9 Proven CI/CD Best Practices for Small Dev Teams"

Each item: subheading + 2-4 sentences + optional example
```

### Ultimate Guide / Pillar Page (SEO cornerstone)
```
Title: "The Complete Guide to [Topic]"
Length: 3,000-5,000+ words
Structure: Table of contents → deep sections → internal links to related posts
Goal: rank for head term, link to cluster posts
```

### Case Study
```
Challenge:   [What was the customer's problem?]
Approach:    [What did they do / how did they use the product?]
Results:     [Specific metrics — "40% faster deploy cycles", "3x fewer bugs"]
Quote:       [Pull quote from customer]
CTA:         [Similar results for you?]
```

## SEO Content Checklist

```
Keyword research:
[ ] Target keyword has search volume (verify in Ahrefs / Semrush / Google Keyword Planner)
[ ] Keyword difficulty matches domain authority (don't chase DA 90 terms with a new site)
[ ] Search intent matches content type (informational → blog, transactional → landing page)

On-page:
[ ] Keyword in title tag (< 60 chars), meta description (< 160 chars), H1
[ ] LSI keywords and synonyms used naturally throughout
[ ] Images have keyword-relevant alt text
[ ] Internal links to 2-3 related posts
[ ] External links to authoritative sources

Structure:
[ ] Table of contents for posts > 1,500 words
[ ] Short paragraphs (2-4 lines)
[ ] Subheadings every 300-400 words
[ ] FAQ section targets "People Also Ask" snippets
```

## Content Calendar Template

```markdown
# Content Calendar — [Month Year]

| Publish Date | Title | Format | Funnel Stage | Keyword | Author | Status |
|---|---|---|---|---|---|---|
| [date] | [title] | Blog | TOFU | [keyword] | [name] | Draft |
| [date] | [title] | Newsletter | Retention | — | [name] | Scheduled |
| [date] | [title] | Case Study | BOFU | [keyword] | [name] | Review |
```

**Sustainable cadence:**
- Solo creator: 1 blog/week + 1 newsletter/week
- Small team: 3 blogs/week + newsletter + 1 long-form/month
- Full team: daily content across channels

Quality > quantity. 1 excellent post/week beats 5 mediocre posts.

## Newsletter Best Practices

```
Subject line:      < 50 chars, no spam words, test with A/B
Send time:         Tuesday–Thursday, 9-11am (recipient's timezone) — test your audience
Unsubscribe:       Always one-click — legal requirement (CAN-SPAM, GDPR)
List hygiene:      Remove bounces + 90-day inactives quarterly

Content structure:
  1. Hook (1-2 sentences — why this email, why now)
  2. Main content (one topic done well, not three topics done poorly)
  3. CTA (one action — read, watch, try, share)
  4. PS (second CTA or teaser for next issue)
```

## Content Distribution Checklist (per piece)

```
Owned channels:
[ ] Published on blog / website
[ ] Shared to email list
[ ] Posted to LinkedIn (+ personal profile repost)
[ ] Tweeted / Xed with key excerpt

Earned / community:
[ ] Shared in relevant Slack / Discord communities (no spam — provide value)
[ ] Submitted to relevant newsletters / roundups
[ ] Repurposed: blog → Twitter thread, blog → LinkedIn carousel

Paid (if budget):
[ ] Promoted post targeting lookalike audience of existing readers
[ ] Retargeting campaign to website visitors who didn't convert
```

## Metrics to Track

| Metric | Tool | Good Baseline |
|---|---|---|
| Organic traffic | Google Search Console / GA4 | Growing MoM |
| Keyword rankings | Ahrefs / Semrush | Top 10 for target terms |
| Email open rate | Mailchimp / ConvertKit | > 30% for engaged list |
| Email CTR | Mailchimp / ConvertKit | > 3% |
| Time on page | GA4 | > 3 min for long-form |
| Backlinks earned | Ahrefs | Growing per month |
| Leads from content | CRM attribution | > 10% of total leads |
