# Copywriter — Copy Frameworks & Writing Reference

Load when writing ads, emails, landing pages, or any persuasive copy.

---

## Core Copy Frameworks

### AIDA
```
Attention  → Grab attention with a bold headline or striking stat
Interest   → Explain why this matters to the reader
Desire     → Paint the picture of life after using the product
Action     → One clear CTA — what to do next
```

### PAS (Problem-Agitate-Solution)
```
Problem   → Name the pain the reader already feels
Agitate   → Intensify it — what happens if they don't solve it?
Solution  → Present your product as the clear answer
```
Best for: ads, email subject lines, short-form social.

### StoryBrand (7-part framework)
```
1. A character (the customer)
2. has a problem (villain/obstacle)
3. and meets a guide (your brand)
4. who gives them a plan
5. and calls them to action
6. that ends in success
7. and helps them avoid failure
```
Best for: homepage hero, brand narrative, onboarding sequences.

### Before-After-Bridge
```
Before: Here's the world with the problem (current pain)
After:  Here's the world without the problem (desired outcome)
Bridge: Here's how to get there (your product)
```
Best for: video scripts, landing pages, case study summaries.

## Headline Formulas

### Specific benefit
`[Do X] in [timeframe] without [obstacle]`
`"Launch your first campaign in 10 minutes — no design skills needed"`

### Curiosity / open loop
`"The [unexpected thing] that [desired outcome]"`
`"The three-word email that doubled our open rate"`

### Social proof
`"How [customer type] [achieved result] with [product]"`
`"How 4,000 startups ship features faster with CTS"`

### Problem + promise
`"Tired of [pain]? [Product] [outcome]"`
`"Tired of rewriting the same prompt every session? CTS remembers."`

### Question
`"[Do you struggle with X]?"`
— Use sparingly. Works best when the yes is obvious.

## CTA Copy Rules

**What makes a strong CTA:**
- Action verb: "Start", "Get", "Download", "Try", "Book"
- Specific: "Start free trial" > "Sign up"
- Low friction: "No credit card required" adjacent to button
- First person: "Start my free trial" > "Start your free trial" (5–15% lift)
- Single CTA per page / email — one job

**Weak → Strong rewrites:**
```
"Submit"                → "Get my free quote"
"Learn more"            → "See how it works"
"Sign up"               → "Start building — it's free"
"Click here"            → "Download the checklist"
"Contact us"            → "Talk to a specialist"
```

## Email Copy Checklist

```
Subject line:
[ ] < 50 characters (mobile preview)
[ ] No spam trigger words (FREE!!!, URGENT, guaranteed)
[ ] Passes the "from a friend" test — does it sound human?
[ ] One idea only — don't summarize the whole email

Preview text:
[ ] Extends the subject line, doesn't repeat it
[ ] < 100 characters

Body:
[ ] Opens with the reader's problem, not about you
[ ] Short paragraphs (2-3 lines max)
[ ] One CTA — not three

From name:
[ ] Person's name, not company name for high-stakes sequences
[ ] e.g., "Alice from CTS" not "CTS Team"
```

## Voice & Tone Guide Template

```markdown
## Brand Voice

**Personality**: [3 adjectives — e.g., "Direct, warm, expert"]

**We are:**         [positive trait]
**We are not:**     [trap to avoid]

Examples:
We are:    Confident  |  We are not: Arrogant
We are:    Simple     |  We are not: Dumbed-down
We are:    Human      |  We are not: Casual/sloppy

## Tone by Context

| Situation | Tone | Example |
|-----------|------|---------|
| Onboarding | Encouraging, clear | "You're all set. Here's your first step." |
| Error messages | Apologetic, helpful | "Something went wrong. Here's what to try." |
| Marketing | Confident, inspiring | "Ship 10x faster. Without the chaos." |
| Support | Patient, empathetic | "We get it — this is frustrating. Let's fix it." |
```

## Readability Rules

```
Flesch-Kincaid Grade: aim for 7-9 (easily readable by most adults)
Sentence length: average < 20 words
Paragraph length: 2-4 lines max (online)
Active voice: "We built X" not "X was built by us"
Contractions: use them — "you're" reads warmer than "you are"
Jargon: define or cut it — if you need to explain it, question whether it belongs
```

## UX Writing Patterns

### Empty state
```
[Icon]
[Friendly headline — what they'll find here]
[Benefit of the action]
[CTA Button]

Example:
"No campaigns yet"
"Create your first campaign to reach your audience."
[Create campaign]
```

### Tooltip
Max 1 sentence. Plain language. Starts with a verb or noun, not "This is...".

### Confirmation dialog
```
[What will happen — specific]  "Delete 'Q3 Campaign'?"
[Consequence if irreversible]  "This cannot be undone."
[Cancel]  [Confirm — red button with action verb] "Delete"
```

### Error messages
```
What went wrong + how to fix it.
Bad:  "Invalid input"
Good: "Enter a password with at least 8 characters and one number."
```
